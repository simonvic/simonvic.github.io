Setup

Copy the image you want to make into an .edds file inside your mod directory

Import the resource

With the workbench, from the resource browser (left), right click on your image and select Register resource and Import

Optimize image

  1. Open yourImage.edds with the Workbench
  2. Select the Import Settings tab on the bottom right of the Workbench
    If the .meta file is not correct (e.g. doesn't have the same name of .edds) you won't be able to see the Import Settings
  3. Set the SourceFile to the image you want to use.
    If you can't select your image, be sure to have it placed in the same directory of the .edds
  4. Change the settings as you need.
    Smaller files are always better. Set FormatCompress to Fastest for smaller file size, and change it if you need more quality.

    If the image is in greyscale, set Conversion to DXTCompression to have the smallest file size.

    The image size has to be power of 2 (128x128, 256x256, 512x512, 1024x1024 etc.) for this to work
  5. Click on Reimport resource(PC)
    The image may not be visible immediately; with the Worbench click, on another file from the ResourceBrowser and reclick on yourImage.edds; you should now be able to see your image

Cleanup

If the .eeds has been created correctly, you can now remove your original image (.png, .targa) from your mod directory to avoid accidentally packing it when building the mod.
