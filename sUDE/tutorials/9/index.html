  1. Setup

    Add sFramework as a dependency by adding it to the requiredAddons inside your config.cpp 

    
class CfgPatches {
	class NAME_OF_YOUR_MOD {
		requiredAddons[] = {
			"sFramework"
		};
	};
};

  2. Definition of a color

    sColor is just a wrapper for an integer value, so it has a minimal memory footprint.

    It enables you to define a color in various way.

    • Hexadecimal value, like in CSS 
      
SColor.rgb(0xFF0000);    //red
SColor.rgba(0xFF000055); //red slightly transparent
SColor.argb(0x55FF0000); //red slightly transparent
    • Separated red, green, blue (and alpha) channels; from 0 to 255 
      
SColor.rgb(60, 97, 178);      //blueish
SColor.rgba(60, 97, 178, 0);  //blueish
SColor.argb(0, 60, 97, 178);  //blueish
    • HSB format (hue, saturation, brightness) 
      
SColor.hsb(0.60, 0.65, 0.87); //yellowish
    • Via presets 
      
SColor.rgb(RGBColors.RED);
SColor.rgb(RGBColors.AQUAMARINE);
SColor.rgb(RGBColors.YELLOW_GREEN);
