  1. Setup

    Add sVisual as a dependency by adding it to the requiredAddons inside your config.cpp 

    
// more stuff here...
class CfgPatches {
	class YourMod {
		// more stuff here...
		requiredAddons[] = {
			"sVisual"
		};
		// more stuff here...
	};
};
// more stuff here...

  2. Override defaults

    When there are no active PPEffects, the PPEManager will apply the default values, which are taken from the config.cpp.
    Simply specify the effect of which you wish to modify the default value, and the value you want to use.
    All you have to do is to override the default values in the cfg_sUDE Visual PPEffects DefaultValues class. To do that simply add the following in your config.cpp 

    
// more stuff here...
class Cfg_sUDE {
	class Visual {
		class PPEffects {
			class DefaultValues {
				saturation = 5.0;
                vignette_strength = 0.5;
				//any other your overridden values here
			};
		};
	};
};
// more stuff here...
